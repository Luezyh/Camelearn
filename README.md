# Camelearn

  Camelearn e uma plataforma que monta um ambiente para o aluno e o professor, oferecendo para o aluno uma boa organização de suas atividades e para os professores uma boa comunicação com os alunos 

## Começando

  Para executar o projeto localmente, verifique se sua máquina contém os seguintes requisitos:
  - Node.js (v25.9.0)
  - MySQL (8.4 LTS)

### Clone o repositório
